# Caesar Cipher Toolkit

A lightweight Python implementation of the classic Caesar cipher algorithm for basic text encryption and decryption.

## 🔐 Logic

- Encrypt text using a Caesar cipher
- Decrypt Caesar-encrypted text
- Preserves uppercase and lowercase letters
- Input validation for shift values (must be integer between 1 and 25)
- Handles non-alphabet characters without modification

## 📦 How It Works

The Caesar cipher shifts each letter in the alphabet by a fixed number of positions. For example, a shift of 3 turns:

- A → D  
- B → E  
- C → F
